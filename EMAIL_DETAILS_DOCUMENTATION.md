## 🔧 **How It Works**

### Booking Flow:
```
1. User books flight on your website
   ↓
2. Booking saved to database
   ↓
3. ✉️ EMAIL AUTOMATICALLY SENT with ALL details
   ↓
4. User receives beautiful confirmation email
   ↓
5. Success message shown on website
```

### Cancellation Flow:
```
1. User cancels booking
   ↓
2. Booking status updated to 'cancelled'
   ↓
3. ✉️ CANCELLATION EMAIL AUTOMATICALLY SENT
   ↓
4. User receives detailed cancellation email with refund info
   ↓
5. Cancellation confirmed on website
```
